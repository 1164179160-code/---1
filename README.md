# 极速骑行 - 高端山地车骑行APP

## 功能特性

✅ **英雄海报区** - 全屏大海报，视差滚动效果  
✅ **车型入口** - 3种车型展示，带视差动画  
✅ **爆款推荐** - 热销车型 + 新品上市  
✅ **配套商品** - 手套、手表、眼镜等6类装备  
✅ **Feed流推荐** - 车型推荐、热门话题、产品服务混合内容  

## 技术实现

- 纯原生 HTML + CSS + JavaScript
- 无需 Node.js 或任何构建工具
- 视差滚动效果
- 3D 卡片悬停动画
- 响应式设计
- 平滑滚动导航

## 使用方法

直接双击 `index.html` 在浏览器中打开即可！

或者在终端运行：
```bash
open index.html
```

```
.

```

## 特色效果

1. **视差滚动** - 英雄区背景和车型卡片都有视差效果
2. **3D悬停** - 鼠标悬停卡片时有 3D 倾斜效果
3. **渐进动画** - 滚动到视口时元素逐个淡入
4. **鼠标光效** - 鼠标移动时有光晕跟随效果
5. **响应式布局** - 自适应手机、平板、桌面端
