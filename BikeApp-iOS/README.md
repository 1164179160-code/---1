# BikeApp - iOS版 (SwiftUI)

## 使用方法

1. 打开 Xcode
2. 创建新项目：File → New → Project
3. 选择 iOS → App
4. 项目名称：BikeApp
5. Interface 选择：SwiftUI
6. 将 `ContentView.swift` 和 `BikeApp.swift` 的内容复制到对应文件
7. 运行项目（Cmd + R）

## 功能特性

- ✅ 底部 Tab 导航
- ✅ 英雄海报区
- ✅ 爆款推荐横向滚动
- ✅ 车型网格展示
- ✅ 配件商城
- ✅ Feed 流推荐

## 系统要求

- macOS 13.0+
- Xcode 15.0+
- iOS 17.0+
