# 🚴 山地车 APP - 快速预览指南

## 方案 1：iOS APP（需要 Xcode）

### 步骤：
1. 打开 App Store
2. 搜索并安装 "Xcode"（约 15GB，需要 30-60 分钟）
3. 安装完成后：
   - 打开 Xcode
   - File → New → Project
   - 选择 iOS → App
   - Interface 选择 SwiftUI
   - 项目名：BikeApp
4. 把 `BikeApp-iOS/ContentView.swift` 的代码复制到项目里
5. 点击运行按钮（▶️）

## 方案 2：在线预览（立即可用）

访问这些在线工具，粘贴代码即可预览：

### React Native 版本：
1. 访问：https://snack.expo.dev
2. 把 `BikeApp-ReactNative/App.tsx` 的代码粘贴进去
3. 点击右侧的手机预览
4. 或扫码在真机上预览

### Flutter 版本：
1. 访问：https://dartpad.dev
2. 把 `BikeApp-Flutter/lib/main.dart` 的代码粘贴进去
3. 点击 Run 按钮

## 方案 3：网页版（已经可用）

直接在浏览器打开：
```bash
open index.html
```

在手机浏览器打开也能用，体验和 APP 一样。
